# 全球风向盘产品说明

## 一句话定位

让用户在 30 秒内看到全球变化趋势的宏观仪表盘。

## 不做什么

- 不做新闻日报
- 不做热点榜
- 不做长篇趋势报告
- 不做按国家平铺的国际新闻
- 不做复杂评分展示

## 做什么

将世界拆成几个稳定系统层，每个系统层每天只输出一个最重要的风向判断。

## 前台展示原则

首页信息少，但每条都必须有判断。

每个观察窗：

```text
领域名
今日风向标题
一句话判断
关键词：3-5 个
```

## 后续可扩展

- 接入 GitHub Actions 定时生成
- 增加详情页
- 增加趋势历史
- 增加盲区提醒算法
- 增加来源证据折叠面板
